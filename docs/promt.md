## Типографіка

Зроби дизайн сильним через типографіку.

Використай максимум 2 шрифти:

1. Основний сучасний sans-serif для тексту і заголовків:

   * Space Grotesk
   * Manrope
   * Inter
   * IBM Plex Sans
   * Satoshi, якщо доступний

2. Моноширинний шрифт для технічних акцентів:

   * JetBrains Mono
   * IBM Plex Mono
   * Fira Code
   * Geist Mono

Рекомендована комбінація:

* `Space Grotesk` для заголовків і важливих текстових акцентів
* `JetBrains Mono` для labels, code-блоку, маленьких технічних елементів
* системні fallback-шрифти, якщо web font не завантажиться

Важливо:

* максимум 2 font-family
* максимум 2–3 font-weight
* додай `font-display: swap`, якщо використовуєш web fonts
* не підключай багато зайвих стилів
* сайт має залишатися легким
* якщо використовуєш Google Fonts, підключай тільки потрібні шрифти і ваги
* якщо можна зробити якісно без CDN — використовуй системні fallback-шрифти

## Кольори

Темна тема за замовчуванням.

Не використовуй банальні кислотні кольори. Палітра має бути спокійна:

* background: майже чорний, але не чистий `#000`
* surface: темно-сірий / slate
* text: м’який білий
* muted text: сірий
* accent: один основний колір, наприклад cold blue, muted violet або emerald
* border: тонкі напівпрозорі лінії

Використовуй accent color дозовано:

* кнопки
* active links
* маленькі labels
* hover states
* важливі слова

Не роби весь сайт градієнтним. Градієнти мають бути тільки як легкий акцент.

## README.md

Обов’язково створи README.md з інструкцією:

1. Як відкрити сайт локально
2. Як змінити ім’я, опис, навички, посилання і соцмережі
3. Як змінити кольори, шрифти і дизайн-акценти
4. Як захостити на VPS через Nginx
5. Приклад конфігурації Nginx
6. Як оновлювати сайт через Git
7. Як підключити HTTPS через Let’s Encrypt

Додай приклад деплою:

```bash
sudo apt update
sudo apt install nginx git -y

cd /var/www
sudo git clone https://github.com/ZaGOR-1/mysitevizitka.git portfolio
sudo chown -R www-data:www-data /var/www/portfolio
```

Приклад Nginx-конфігу:

```nginx
server {
    listen 80;
    server_name about.me.hotzagor.tech;

    root /var/www/portfolio;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }
}
```

І команди:

```bash
sudo ln -s /etc/nginx/sites-available/portfolio /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

Інструкція для HTTPS через Let’s Encrypt:

```bash
sudo apt install certbot python3-certbot-nginx -y
sudo certbot --nginx -d about.me.hotzagor.tech
```
